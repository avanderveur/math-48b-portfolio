Exponential Functions + Repeated Multiplication

Exponentials model repeated multiplication and rapid growth/decay. Their graphs have a horizontal asymptote and change by a constant ratio.

Core Idea

An exponential function has the form:

f(x) = a · bx
1

a is the starting value (y-intercept).

2

b is the growth/decay factor (multiplier each step).

3

If b > 1 → growth. If 0 < b < 1 → decay.

Repeated Multiplication Pattern

Every time x increases by 1, the output is multiplied by b.

f(x+1) = f(x) · b

Summary

Exponentials change by a constant multiplier. Growth increases away from 0, decay decreases toward 0, and both approach a horizontal asymptote.
