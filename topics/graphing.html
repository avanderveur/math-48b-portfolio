Graphing Rational Functions (Full Analysis)

A full rational graph includes: domain, holes, vertical asymptotes, horizontal asymptotes, intercepts, and correct end behavior.

Full Graphing Checklist

1

Factor numerator and denominator.

2

Find domain restrictions (denominator = 0).

3

Cancel common factors → identify any holes.

4

Vertical asymptotes from remaining denominator factors.

5

Horizontal asymptote from degree rules.

6

Find intercepts and plot key points.

7

Sketch a correct curve approaching asymptotes.

Example 1 (Animated + Real Graph)

f(x) = (x + 2) / (x − 1)
Press Next or Play to start.
Denominator = 0 → x ≠ 1
VA: x = 1
HA: y = 1
x-intercept: (−2, 0)
y-intercept: (0, −2)

Example 2 (Animated + Real Graph)

g(x) = (x² − 4) / (x² − 5x + 6)
Press Next or Play to start.
Factored: (x−2)(x+2) / [(x−2)(x−3)]
Domain: x ≠ 2, 3
Hole: (2, −4)
VA: x = 3
HA: y = 1

Summary

A full rational graph is built from restrictions, asymptotes, intercepts, holes, and correct curve behavior. The graph should always reflect the algebra.
